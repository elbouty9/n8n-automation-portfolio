~30 min

💳 Créer un compte Stripe

Compte gratuit sur stripe.com. Activer le mode test — toutes les opérations seront simulées avec des cartes fictives, aucun vrai paiement. Carte test : 4242 4242 4242 4242.

StripeMode testGratuit

⚡ Webhook Stripe → N8N

Dans Stripe → Developers → Webhooks → ajouter l'URL N8N. Événement à écouter : "payment_intent.succeeded". Stripe enverra toutes les données du paiement à N8N automatiquement.
