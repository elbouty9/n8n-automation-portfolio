Étape 1 — Configurer Google Drive

~30 min
▶

📁 Créer le dossier "Documents à résumer"

Dans Google Drive crée un dossier dédié. C'est ce dossier que N8N va surveiller en permanence — dès qu'un PDF y est déposé le workflow se déclenche.

Google DriveDossier

⚡ Google Drive Trigger dans N8N

Node "Google Drive Trigger" → surveille le dossier → se déclenche sur "File Created". Tu apprendras un nouveau type de trigger basé sur des événements externes.

Google Drive TriggerEvent
📝
Étape 2 — Télécharger et extraire le texte
Download + extraction PDF
~45 min
▶

⬇️ Télécharger le PDF depuis Drive

Node Google Drive → Download File. Tu récupères le fichier binaire du PDF dans N8N pour pouvoir l'analyser.

Google DriveBinary data

📝 Extraire le texte du PDF

Node "Extract from File" → PDF → texte brut. C'est un node natif N8N qui lit le contenu du PDF sans API externe. Tu apprendras à manipuler des fichiers binaires.

Extract from FilePDFBinary
🤖
Étape 3 — Résumé IA avec GPT
Résumé + points clés + action recommandée
~30 min
▶

🤖 GPT — Analyser le document

GPT reçoit le texte extrait et retourne un résumé structuré en JSON : titre, résumé en 3 phrases, 5 points clés, type de document, action recommandée.

OpenAIJSON structuré

⚙️ Parser le JSON

Code node pour extraire proprement les champs du JSON GPT — comme on a fait sur les projets précédents. Tu maîtrises déjà ça !

📓 Créer une page Notion

Node Notion → Create Page dans une database. La page contiendra : titre du document, résumé, points clés, date, lien Google Drive. Tu apprendras le node Notion qui est très demandé.

NotionDatabasePage
