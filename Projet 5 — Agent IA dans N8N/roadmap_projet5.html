🧠 C'est quoi un AI Agent ?

Contrairement à GPT qui répond juste à une question, un AI Agent peut utiliser des outils, prendre des décisions et enchaîner des actions de façon autonome. Il décide lui-même quoi faire selon le contexte — c'est le vrai saut vers l'automatisation intelligente.

AI AgentNouveau !Concept clé

⚙️ Configurer le node AI Agent dans N8N

N8N a un node natif "AI Agent" qui permet de connecter un LLM (GPT) avec des outils. Tu lui donnes un rôle, des instructions et des outils disponibles — il choisit lui-même lesquels utiliser selon chaque situation.

🧠 Window Buffer Memory

Sans mémoire l'agent oublie tout entre chaque email. Avec le node "Window Buffer Memory" il garde le contexte des N derniers échanges — il peut donc faire le lien entre les emails d'un même thread.

🎯 System prompt — Le cerveau de l'agent

Le system prompt définit qui est l'agent, ce qu'il peut faire et ses règles de décision. C'est la partie la plus importante — un bon system prompt fait toute la différence entre un agent utile et un agent inutile.
