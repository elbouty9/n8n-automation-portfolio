🤖 Workflow N8N — Prospection Automatisée

Clique sur chaque étape pour voir les détails, outils et conseils

1
🔍 Scraping & Collecte de Leads
Source

On récupère des leads depuis une ou plusieurs sources selon ta cible.

Outils N8N

HTTP Request, Apify node, Phantombuster via webhook

Sources recommandées

LinkedIn Sales Nav, Apollo.io, Google Maps (pour locaux)

Données à collecter

Nom, poste, entreprise, email, LinkedIn URL, taille boîte

Conseil débutant

Commence avec Apollo.io — il a une API gratuite simple à brancher

2
🔄 Déduplication AJOUT
Qualité

Avant tout traitement, on vérifie si le lead existe déjà dans le Google Sheet pour éviter les doublons et les relances embarrassantes.

Nœuds N8N

Google Sheets (lookup) → IF node → Stop si doublon

Clé de dédup

Email (le plus fiable) ou LinkedIn URL

3
🧩 Enrichissement AJOUT
Données+

On enrichit le profil avant de le scorer — un bon score nécessite de bonnes données.

Outils

Hunter.io (email), Clearbit (gratuit limité), LinkedIn scrape

Données utiles

Levée de fonds récente, tech stack, offres d'emploi actives, news

Signaux d'achat

Recrutement actif = budget disponible. Nouveau poste = opportunité.

N8N tip

Chaîne HTTP Request → Set node pour structurer les données proprement

4
⭐ Scoring du Lead
IA / Logique

On attribue un score de 1 à 100 selon des critères pondérés. Deux approches possibles.

Approche règles (débutant)

IF nodes en cascade : poste = CEO → +30pts, taille 10-50 → +20pts, etc.

Approche IA (recommandée)

GPT-4 via OpenAI node avec prompt structuré retournant un JSON avec score + raison

Critères clés

Titre/séniorité, taille entreprise, secteur, signaux d'achat, localisation

Seuil de coupure

Score < 40 → archivé sans message. Score ≥ 40 → on continue.

5
✍️ Génération du Message
IA

On génère un message hyper-personnalisé basé sur toutes les données collectées.

Nœud N8N

OpenAI node (GPT-4o) avec prompt template utilisant les variables du lead

Structure du message

Hook personnalisé → Problème identifié → Valeur proposée → CTA clair

Personnalisation

Mentionner un article récent, une levée de fonds, un recrutement spécifique

Variantes

Génère 2-3 versions avec des tons différents → le scorer choisit

6
🎯 Scoring du Message
Qualité

On évalue chaque message généré avant de le valider — pas question d'envoyer quelque chose de moyen.

Critères d'évaluation

Personnalisation réelle, clarté du CTA, longueur, absence de spam words

Seuil

Score < 70 → régénère avec un prompt amélioré (loop). Score ≥ 70 → validé.

Anti-boucle infinie

Max 3 tentatives, ensuite → file d'attente manuelle pour review humaine

N8N tip

Utilise un Loop node ou un compteur dans le Set node pour limiter les itérations

7
💾 Stockage & Suivi AMÉLIORÉ
Output

On stocke tout dans Google Sheets avec une structure pensée pour le suivi et la boucle de feedback.

Colonnes essentielles

Date, Nom, Email, Score lead, Score message, Message final, Statut, Réponse reçue

Boucle de feedback

Si réponse reçue → webhook → met à jour le statut → nourrit ton prompt de scoring

Canaux d'envoi optionnels

Gmail node, Lemlist via API, ou simplement colonne "À envoyer" manuellement

Bonus

Ajoute un onglet "Statistiques" avec formules pour tracker ton taux de réponse

💡 Par où commencer ? Commence par l'étape 1 → 4 → 7 seulement. Un workflow simple qui fonctionne vaut mieux qu'un workflow parfait qui ne tourne pas. Ajoute les étapes 2, 3, 5, 6 une par une ensuite.
