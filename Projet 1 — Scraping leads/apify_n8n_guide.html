🗺️ Apify Google Maps → N8N

Guide interactif — coche chaque étape au fur et à mesure

✓
1. Récupère ta clé API Apify
Apify

Va sur apify.com → ton compte → Settings → Integrations → API tokens

Crée un token et copie-le, tu en auras besoin dans N8N.

💡 Le plan gratuit Apify donne ~5$ de crédits/mois, suffisant pour scraper ~500-1000 établissements Google Maps.

Ensuite, cherche l'acteur "Google Maps Scraper" dans le store Apify (par Compass). C'est le plus fiable et le mieux maintenu.

✓
2. Configurer l'acteur dans Apify
Apify

Dans l'acteur Google Maps Scraper, configure l'input JSON comme suit :

{ "searchStringsArray": ["agences marketing Paris"], "maxCrawledPlacesPerSearch": 50, "language": "fr", "countryCode": "fr", "exportPlaceUrls": false, "additionalInfo": true }
⚠️ Change "agences marketing Paris" par ta vraie recherche. Tu peux mettre plusieurs termes dans le tableau.

Lance un test rapide avec maxCrawledPlacesPerSearch: 5 pour vérifier que ça tourne avant de brancher N8N.

Les champs importants que tu vas récupérer :

Champ ApifyCe que c'est
titleNom de l'établissement
addressAdresse complète
phoneTéléphone
websiteSite web
totalScoreNote Google (sur 5)
reviewsCountNombre d'avis
categoryNameCatégorie (ex: Agence marketing)
✓
3. Créer le workflow N8N — nœud déclencheur
N8N

Dans N8N, crée un nouveau workflow. Pour commencer, utilise un Manual Trigger (tu cliqueras manuellement pour lancer).

Plus tard tu pourras le remplacer par un Schedule Trigger (ex: tous les lundis à 8h).

💡 N8N Cloud (gratuit jusqu'à 5 workflows actifs) ou N8N en local via Docker. Les deux fonctionnent pareil.
✓
4. Nœud HTTP Request → lancer le scraping
N8N

Ajoute un nœud HTTP Request et configure-le ainsi :

ParamètreValeur
MethodPOST
URLhttps://api.apify.com/v2/acts/compass~crawler-google-places/runs
AuthenticationHeader Auth
Header NameAuthorization
Header ValueBearer TON_API_TOKEN
Content-Typeapplication/json

Body (JSON) :

{ "searchStringsArray": ["agences marketing Paris"], "maxCrawledPlacesPerSearch": 50, "language": "fr", "countryCode": "fr" }
⚠️ Ce nœud lance le scraping mais ne retourne pas encore les données — Apify travaille en asynchrone.

La réponse contiendra un runId que tu utiliseras à l'étape suivante.

✓
5. Attendre la fin + récupérer les résultats
N8N

Apify prend du temps. Voici la stratégie la plus simple pour débutant :

Option A (simple) : Ajoute un nœud Wait de 60 secondes après le lancement, puis un deuxième HTTP Request pour récupérer les données.

Option B (robuste) : Utilise un webhook Apify → configure dans Apify un webhook qui appelle ton N8N quand le run est terminé.

💡 Pour apprendre, commence par l'Option A. C'est plus simple à déboguer.

Nœud Wait → durée : 60 secondes

Puis un nouveau HTTP Request pour récupérer les résultats :

ParamètreValeur
MethodGET
URLhttps://api.apify.com/v2/acts/compass~crawler-google-places/runs/last/dataset/items
Query param: tokenTON_API_TOKEN
Query param: limit50

Tu récupères un tableau JSON avec tous tes leads 🎉

✓
6. Normaliser les données avec un Set node
N8N

Ajoute un nœud Set pour renommer les champs Apify en champs standardisés. Ça facilitera l'ajout d'Apollo plus tard.

// Mapping dans le Set node nom → {{ $json.title }} adresse → {{ $json.address }} telephone → {{ $json.phone }} site_web → {{ $json.website }} note → {{ $json.totalScore }} nb_avis → {{ $json.reviewsCount }} categorie → {{ $json.categoryName }} source → "apify_gmaps" date_scrape → {{ $now }}
💡 Le champ source fixé à "apify_gmaps" te permettra de savoir d'où vient chaque lead dans ton Google Sheet.
✓
7. Stocker dans Google Sheets
N8N

Ajoute un nœud Google Sheets en mode "Append or Update Row".

Prépare ton Google Sheet avec ces colonnes :

ColonneValeur N8N
Nom{{ $json.nom }}
Adresse{{ $json.adresse }}
Téléphone{{ $json.telephone }}
Site Web{{ $json.site_web }}
Note Google{{ $json.note }}
Nb Avis{{ $json.nb_avis }}
Catégorie{{ $json.categorie }}
Source{{ $json.source }}
Date Scrape{{ $json.date_scrape }}
Score Lead(vide pour l'instant)
Message(vide pour l'instant)
StatutNouveau
⚠️ Pour la dédup : utilise "Matching Columns" = Téléphone ou Site Web. N8N mettra à jour si ça existe déjà au lieu de créer un doublon.

🎉 Bravo, ton pipeline Apify → N8N → Google Sheets est prêt !

Prochaine étape : ajouter le scoring GPT entre le Set node et Google Sheets, puis brancher Apollo.io en deuxième source avec un Merge node.

Tu peux relancer le guide à tout moment en cochant/décochant les étapes.
