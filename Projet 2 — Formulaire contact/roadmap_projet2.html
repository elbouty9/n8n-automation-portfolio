Étape 1 — Créer le formulaire Tally
Gratuit · 10 minutes
~30 min
▶

📝 Créer un compte Tally.so

Va sur tally.so et crée un compte gratuit. Crée un formulaire avec : Nom, Email, Téléphone, Type de demande (liste déroulante), Message.

Tally.soGratuit

⚡ Brancher le Webhook N8N

Dans Tally → Integrations → Webhooks → colle l'URL du Webhook node N8N. Chaque soumission enverra les données directement à N8N.

WebhookN8N
🔀
Étape 2 — Triage et qualification IA
IF node + GPT
~1h
▶

🔀 IF node — Triage par type

Si "Type de demande" = Urgent → priorité haute. Si = Devis → pipeline commercial. Si = Support → ticket support. Tu apprends à router différemment selon les données.

IF nodeRouting

🤖 GPT — Résumé de la demande

GPT lit le message et retourne un résumé en 1 phrase + niveau de priorité + action recommandée. Parfait pour les équipes qui reçoivent beaucoup de demandes.

OpenAIRésuméPriorité

Étape 3 — Notifications Gmail + Slack
Les deux canaux les plus demandés
~1h
▶

📧 Gmail — Email formaté

Envoie un email HTML propre avec toutes les infos du prospect + résumé GPT + bouton "Répondre". Tu apprends le node Gmail et le formatage HTML dans N8N.

GmailHTML

💬 Slack — Notification équipe

Envoie un message Slack formaté avec les infos clés. Si urgent → mention @channel. Tu apprends le node Slack et les messages conditionnels.

SlackMentionsConditionnel

📊 Google Sheets — Archive toutes les demandes

Tu connais déjà ce node — facile ! Date, nom, email, type, résumé GPT, priorité, statut.

Google Sheets

✅ Email de confirmation au prospect

GPT génère un email de confirmation personnalisé envoyé automatiquement au prospect. "Merci [Prénom], nous avons bien reçu votre demande concernant [sujet]..."
